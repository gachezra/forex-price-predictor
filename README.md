# Forex Price Predictor
Uses tensors to map out forex data (h,l,o,c) and pin point the patterns then map out the next close in a candle stick. Doesn't give an accurate prediction, but at least it works.

## Tech Stack
1. TensorFlow js
2. Node js
3. Chart js
